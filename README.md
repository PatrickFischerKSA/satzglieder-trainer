# Satzglieder-Trainer

Eigenständiges Projekt (ohne Abhängigkeit zu anderen Repos).

## Features

- Sofortkorrektur pro Antwort
- Erklärfunktion bei Fehlern
- Steigender Schwierigkeitsgrad
- Repetitionsmodus bei Fehlern
- Automatisch generierte Übungssätze

## Start lokal

Einfach `index.html` im Browser öffnen.

Optional mit lokalem Webserver:

```bash
cd satzglieder-trainer
python3 -m http.server 8080
```

Dann: <http://localhost:8080>
